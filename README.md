## Getting Started
Go on your project folder
```bash
$ git clone https://github.com/oncletim/pokemon-store-vuejs.git
$ cd pokemon-store-vuejs
$ yarn install 
$ yarn server
```
Go to http://localhost:8080/



## Built With
- Vue js
- VueX (state management)
- Vue Router
